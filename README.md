## 🛠️ KEY FEATURES

### 🛰️ Multi-Tab Architecture
A professional trapezoid-based tab system allowing for multiple concurrent proxy sessions. Each tab maintains its own state, history, and metadata.

### 🔍 Real-Time Metadata Syncing
The address bar and tab titles update in real-time as you navigate. Galactic automatically decodes technical proxy paths back into clean, readable URLs (e.g., `https://google.com`).

### 🎮 Stealth Node Hub (`galactic://games`)
A built-in hub containing **197+ pre-configured nodes**. Games are launched via an `about:blank` stealth cloaking mechanism to bypass `X-Frame-Options` and `CORS` security blocks.

### 🛡️ Safari Optimization
Includes specific "MIME-fix" headers and Service Worker bypass filters to ensure high-speed WebAssembly execution on iOS and MacOS devices.

## 🚀 DEPLOYMENT (STORMKIT)

Galactic is optimized for **Stormkit.io** hosting via **GitHub**.

1.  **Repository Setup:**
    *   Upload `src/`, `public/`, and `package.json` to your repo.
    *   Ensure your 197 games are inside `public/`.
2.  **Stormkit Connection:**
    *   Connect your GitHub account to Stormkit.
    *   Select the `Galactic` repository.
3.  **Environment Config:**
    *   **Runtime:** Node.js
    *   **Build Command:** `npm install`
    *   **Start Command:** `npm start`
4.  **Deployment:**
    *   Deploy the main branch. You will receive a permanent `*.stormkit.io` uplink.

1.  Open your Galactic URL.
2.  The system will trigger the **Staggered Entry Animation**.
3.  **MANDATORY:** Refresh the page manually once after the initial load. This allows Safari to properly register the Service Worker under the required security headers.
4.  Once the HUD displays `VANGUARD SYSTEM READY`, the uplink is stable.
